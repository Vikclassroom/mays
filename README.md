# mays-angular

## Lancement

Pour lancer le projet, lancez le backend (mays-dotnet), puis les commandes suivantes dans un terminal

```bash
cd maysAngular
npm install
ng serve    
```

Aller sur un nvaigateur et ouvrir une page localhost:4200
